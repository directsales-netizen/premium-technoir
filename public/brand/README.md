Place your official logo file(s) here.

Required filename for the site to pick it up automatically:

  - `public/brand/logo.png`

Recommended sizes:

  - 512x512 PNG (high resolution, used by social previews)
  - 256x256 PNG (fallback)
  - 120x120 PNG (favicon-ish)

Usage:

  - The site header component expects `/brand/logo.png`.
  - Replace the file with your provided logo image.
